# job4j_di

## DI - внедрение зависимостей

---
### Используемый стек
- Java 17
- Apache Maven 3.8.5
- JUnit5
- AssertJ

---
### Требуемое окружение
- JDK 17
- Apache Maven 3.8.5

---
### Подготовка к запуску приложения
- Собрать jar с приложением, выполнив команду `mvn clean install`
- Запустить приложение, выполнив команду: `java -jar job4j_di`
